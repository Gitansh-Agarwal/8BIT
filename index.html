In this online compiler, you will find several buttons that serve different functions.
The button links are as follows:

"Copy" button: You can use the F8 key to quickly copy the code from the editor.
"Save" button: Pressing F9 will save your code in the compiler.
"Clear All" button: Pressing the F12 key will clear the code editor, removing all the code.
"Run" button : Pressing F5 button will execute the code.
"Console" button : Double tap the button to show console logs and double tap on console window or button to close it
